## Пошаговый мануал для пользователя
# шаг 1-й
Запустите программу лабораторной работы LabARU3_st.exe
# шаг 2-й
В открывшимся окне нажмите далее
# шаг 3-й
В следующем окне ввидите свои данные и нажмите далие
# шаг 4-й
Теперь, запустите программу-бот ARU3_st.exe
# шаг 5-й

Нажмите Enter на вашей клавиатуре, как просит Бот, и идите попейте чайок на ~10мин
# Примечания
* Бот выполняет работу тупыми методами, из-за чего не точно выстовляет переменные генератора. Так что лабу может придется переделывать раза 3
